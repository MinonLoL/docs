# Тепловая карта и дашборд

## Тепловая карта загрузки

Тепловая карта отображает загрузку ПВЗ по часам и дням недели. Цветовая гамма:
- Зелёный: нормальная загрузка (<85%)
- Жёлтый: средняя загрузка (85–100%)
- Красный: перегрузка (100–120%)
- Тёмно-красный: критическая перегрузка (>120%)

Пример использования:
```mermaid
heatmap
    title Загрузка ПВЗ за неделю
    x-axis "Понедельник", "Вторник", "Среда", "Четверг", "Пятница"
    y-axis "08:00", "10:00", "12:00", "14:00", "16:00", "18:00"
    data [
        [0.7, 0.8, 0.9, 1.1, 1.2],
        [0.6, 0.7, 0.8, 1.0, 1.1],
        [0.5, 0.6, 0.7, 0.9, 1.0],
        [0.8, 0.9, 1.0, 1.2, 1.3],
        [0.7, 0.8, 0.9, 1.1, 1.2]
    ]
```

## Дашборд с пороговыми индикаторами

Дашборд отображает ключевые метрики:
- Средняя загрузка в часы пик.
- Доля интервалов с перегрузкой > 100%.
- Общее количество операций за период.

Индикаторы:
- Зелёный: метрика в пределах нормы.
- Красный: превышение порогового значения.
