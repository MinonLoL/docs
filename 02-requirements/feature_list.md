## F-01: Аутентификация и авторизация

- Вход в систему по логину и паролю.
- Три роли: `operator`, `supervisor`, `analyst`.
- Ограничение данных оператора — только его `pvz_id`.
- Декоратор `@role_required` на всех защищённых эндпоинтах.
- **Статус:** ✅ Реализовано

## F-02: Управление ПВЗ и расписанием

- Просмотр справочника ПВЗ: `pvz_id`, `address`, `capacity_per_hour`, `region`.
- Просмотр расписания: `weekday`, `open_time`, `close_time`.
- **Статус:** ✅ Реализовано

## F-03: Регистрация операций

- Создание операции: `pvz_id`, `ts`, `type` (`in` / `out` / `return`).
- Валидация: ПВЗ существует, тип корректен, есть расписание на день, `ts` в рабочих часах.
- При нарушении — запись в `error_log` с полем `reason`.
- **Статус:** ✅ Реализовано

## F-04: Просмотр операций

- Список операций с фильтрацией по дате, ПВЗ, типу.
- Доступ ограничен ролью: оператор видит только свои, супервайзер — регион, аналитик — всё.
- **Статус:** ✅ Реализовано

## F-05: Отчёт «Загрузка ПВЗ»

- Агрегация операций по часовым интервалам.
- Поля: `pvz_id`, `date`, `hour`, `ops_per_hour`, `capacity_per_hour`, `load %`, `overload` (флаг).
- Фильтры: период, ПВЗ, тип операции.
- Подсветка строк с `load > 100%`.
- **Статус:** ✅ Реализовано

## F-06: KPI-панель

- Средняя загрузка в часы пик за выбранный период.
- Доля интервалов с `load > 100%`.
- Количество критических перегрузок (`load > 120%`).
- **Статус:** ✅ Реализовано

## F-07: Тепловая карта

- Визуализация загрузки по осям «день недели» × «час суток».
- Цветовая шкала: зелёный (<85%), жёлтый (85–100%), красный (100–120%), тёмно-красный (>120%).
- Доступна для оператора (своя точка) и супервайзера (любой ПВЗ региона).
- **Статус:** ✅ Реализовано

## F-08: Лог ошибок валидации

- Таблица `error_log`: `pvz_id`, `ts`, `op_type`, `reason`, `logged_at`.
- Фильтрация по ПВЗ и дате.
- Доступ: супервайзер (регион), аналитик (все).
- **Статус:** ✅ Реализовано

## F-09: Экспорт CSV

- Выгрузка отчёта загрузки с применёнными фильтрами.
- Формат: `pvz_id,date,hour,ops,capacity,load`; кодировка UTF-8.
- Доступ: роль `analyst`.
- **Статус:** ✅ Реализовано
