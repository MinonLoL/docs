## Базовый URL

```
http://127.0.0.1:5000
```

## Аутентификация

Все защищённые эндпоинты требуют активной сессии Flask. Сессия создаётся после успешного логина.

---

## Auth

### POST /api/auth/login

Вход в систему.

**Тело запроса:**
```json
{ "username": "operator1", "password": "secret" }
```

**Ответ 200:**
```json
{ "role": "operator", "pvz_id": 1 }
```

**Ответ 401:** Неверные учётные данные.

### POST /api/auth/logout

Выход из системы. Очищает сессию.

## ПВЗ

### GET /api/pvz

Список ПВЗ (оператор видит только свой, супервайзер/аналитик — все или по региону).

**Query params:** `region` (опционально)

**Ответ 200:**
```json
[
  { "pvz_id": 1, "address": "ул. Ленина, 10", "capacity_per_hour": 50, "region": "Центральный" }
]
```

### GET /api/pvz/{pvz_id}/schedule

Расписание конкретного ПВЗ.

**Ответ 200:**
```json
[
  { "weekday": 0, "open_time": "09:00", "close_time": "21:00" }
]
```

## Операции

### GET /api/operations

Список операций с фильтрацией.

**Query params:** `pvz_id`, `date_from`, `date_to`, `type`

**Ответ 200:**
```json
[
  { "op_id": 1, "pvz_id": 1, "ts": "2026-05-12T10:23:00", "type": "out" }
]
```

### POST /api/operations

Создание новой операции.

**Тело запроса:**
```json
{ "pvz_id": 1, "ts": "2026-05-12T10:23:00", "type": "in" }
```

**Ответ 201:**
```json
{ "op_id": 1042, "pvz_id": 1, "ts": "2026-05-12T10:23:00", "type": "in" }
```

**Ответ 400:** Ошибка валидации с полем `error` и описанием причины.

## Отчёты

### GET /api/report/load

Отчёт загрузки ПВЗ по часам.

**Query params:** `pvz_id` (опционально), `date_from`, `date_to`, `type` (опционально)

**Ответ 200:**
```json
[
  {
    "pvz_id": 1,
    "pvz_address": "ул. Ленина, 10",
    "date": "2026-05-12",
    "hour": "10:00",
    "ops_per_hour": 45,
    "capacity_per_hour": 50,
    "load_pct": 90.0,
    "overload": false,
    "status": "Предупреждение"
  }
]
```

### GET /api/report/kpi

KPI-панель по всей сети или региону.

**Query params:** `date_from`, `date_to`, `region` (опционально)

**Ответ 200:**
```json
{
  "avg_peak_load_pct": 78.5,
  "overload_share_pct": 4.2,
  "critical_overload_count": 0,
  "total_operations": 4821
}
```

### GET /api/report/heatmap

Данные для тепловой карты.

**Query params:** `pvz_id` (обязателен), `date_from`, `date_to`

**Ответ 200:**
```json
{
  "pvz_id": 1,
  "days": ["Пн", "Вт", "Ср", "Чт", "Пт", "Сб", "Вс"],
  "hours": ["09:00", "10:00", "11:00", "12:00"],
  "data": [
    [0.5, 0.7, 0.9, 1.1],
    [0.6, 0.8, 1.0, 1.2]
  ]
}
```

### GET /api/report/export/csv

Экспорт отчёта в CSV. Доступен только роли `analyst`.

**Query params:** `pvz_id`, `date_from`, `date_to`, `type`

**Ответ 200:** Файл `report.csv` с заголовком `Content-Disposition: attachment`.

Формат файла:
```
pvz_id,date,hour,ops,capacity,load
1,2026-05-12,10,45,50,0.9
```

## Лог ошибок

### GET /api/errors

Лог ошибок валидации.

**Query params:** `pvz_id` (опционально), `date_from`, `date_to`

**Ответ 200:**
```json
[
  {
    "id": 55,
    "pvz_id": 2,
    "ts": "2026-05-12T07:00:00",
    "op_type": "in",
    "reason": "Операция вне рабочих часов",
    "logged_at": "2026-05-12T10:30:00"
  }
]
```

---

## Коды ответов

| Код | Значение                                         |
|-----|--------------------------------------------------|
| 200 | Успешный запрос                                  |
| 201 | Ресурс создан                                    |
| 400 | Ошибка валидации (подробности в теле ответа)     |
| 401 | Не авторизован                                   |
| 403 | Доступ запрещён для данной роли                  |
| 404 | Ресурс не найден                                 |
| 500 | Внутренняя ошибка сервера                        |
