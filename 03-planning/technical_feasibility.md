## Техническая реализуемость

| Критерий                                    | Оценка     | Обоснование                                                         |
|---------------------------------------------|------------|---------------------------------------------------------------------|
| Backend (Python + Flask)                    | Реализуемо | Команда имеет опыт работы с Flask                                   |
| Frontend (HTML/CSS/JS, Jinja2)              | Реализуемо | Стандартный стек, не требует сложного фреймворка                    |
| БД (SQLite)                                 | Реализуемо | Встроенная БД, не требует отдельного сервера; достаточна для MVP    |
| Авторизация по ролям (сессии Flask)         | Реализуемо | Простая схема без JWT; подходит для учебного проекта                |
| Агрегация загрузки по часам                 | Реализуемо | SQL GROUP BY + strftime; алгоритмически несложно                    |
| Тепловая карта (Chart.js)                   | Риск       | Требует дополнительного изучения библиотеки визуализации            |
| Экспорт CSV                                 | Реализуемо | Стандартный модуль `csv` в Python                                   |
| Валидация 5+ правил                         | Реализуемо | Проверки на уровне API + триггеры на уровне логики                  |
| Генерация синтетических данных (≥200 ops)   | Реализуемо | Скрипт `generate_realistic_data.py` уже реализован                 |

## Организационная реализуемость

| Критерий                     | Оценка     | Обоснование                                                              |
|------------------------------|------------|--------------------------------------------------------------------------|
| Команда (5 человек)          | Достаточно | Покрыты роли: PM, backend ×2, frontend, fullstack                        |
| Сроки (~10 недель)           | Реализуемо | При фокусе на Must-функциональность и параллельной работе                |
| Инструменты                  | Доступно   | Git, Python, SQLite — бесплатные и общедоступные                         |
| Деплой                       | Реализуемо | Локальный запуск; при необходимости — nginx + pm2 на VPS                 |

## Вывод

Проект реализуем в заданные сроки при строгом фокусе на Must-функциональность (US-01–US-04, US-07–US-09, US-12–US-14). Наибольший риск — реализация тепловой карты с Chart.js; для его снижения предусмотрено раннее прототипирование на третьей неделе. Все остальные компоненты стека знакомы команде и имеют достаточную документацию.
